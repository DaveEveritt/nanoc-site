This manual will tell you everything there is to know about nanoc. You may want to read the Tutorial first, especially if you're new to nanoc.

The first chapter, An Introduction to nanoc, explains nanoc's philosphy, why it was created, how it was created, etc. You can skip this chapter if you want; it only provides background information.

The second chapter, Using nanoc, teaches you how use nanoc: how to create sites, how to create pages, how to compile site, how to customize the layout, etc. This chapter is a must-read for everyone.

The third and last chapter, Extending nanoc, tells you how to customize nanoc by writing custom filters, custom layout processors and custom data sources.
