nanoc 2.1.5 is out now. Included in this release is compatibility with Ruby 1.9, and a fix for a bug which would cause the extensions attribute for assets to be ignored completely.

Update your copy of nanoc the usual way by issuing the "gem update" command, like this:

> sudo gem update nanoc

If you have any trouble with this nanoc release, please do let me know on the mailinglist or the IRC channel. Enjoy!
