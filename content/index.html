nanoc is a static site generator, fit for building anything from a small personal blog to a large corporate web site.

The GitHub API web site is written using nanoc. Several others API documentation projects also use nanoc: take a look at the App.net API, the Flattr API and the Spree API, for instance.

Are you interested? Download nanoc and follow the tutorial!

The current release is 3.4.3, released on December 9, 2012. Check out the release notes for details.
