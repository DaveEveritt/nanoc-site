Overview

Nanoc3::CodeSnippet represent a piece of custom code of a nanoc site. It contains the textual source code as well as a mtime, which is used to speed up site compilation.

Instance Attribute Summary

  • - (String) data readonly A string containing the actual code in this code snippet.
  • - (String) filename readonly The filename corresponding to this code snippet.
  • - (Time) mtime readonly The time where this code snippet was last modified.
